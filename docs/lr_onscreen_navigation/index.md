# lr_onscreen_navigation

## AddPosition

```lua
exports.lr_onscreen_navigation:AddPosition(self.name,
  icon --[[ string ]],
  color --[[ {r, g , b} ]],
  coords --[[ vector3 ]],
  function(navigation)
    navigation = navigation
    --[[
      methods:
      SetShow(state)
      SetCoords(coords)
      SetColor(newColor)
      SetIcon(newIcon)
      SetDrawDistance(value)
      Destroy()
    ]]
  end
)
```
