# SettingVariable_GetVar

## Get server setting variables

### Arguments

| Arguments    | Type                   |
| ------------ | ---------------------- |
| key          | string                 |
| defaultValue | string, number, object |

### Return

string, number, object

## Example

```lua
exports['ProjectStarboy']:SettingVariable_GetVar("soap_use_time", 10000)
```
